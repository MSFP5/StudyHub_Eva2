# StudyHub_Eva2
Desarrollar una aplicación móvil para la evaluación 2.
Integrantes: Martina Flores y Benjamin Quezada

# Descripcion del Proyecto
El proyecto consiste en la creacion de una aplicacion movil para una empresa dedicada a la reserva de salas de estudio, conferencias y reuniones. El flujo consiste en poder iniciar sesion y registrar una nueva cuenta. Al momento de inicar te debe permitir vizualizar el horario junto con las salas disponibles a reservar, al momento de realizar la reserva debes incluir una imagen de la persona permitiendo asi reconozarla al momento de ingresar a la empresa.

# Funcionalidades
La aplicacion contiene las diguientes funcionalidades clave que ayudan a la logica del negocio:
- Autentificacion Segura: Iniciar sesion y registrarse.
- Home: Visualizar 
- Reservar salas sehun la fecha y hora actual.
- Generacion de un qr automatico que guarda la infromacion de sala.

# Endpoints Utilizados
La aplicacion contiene una API REST que permite generar qr de forma automatica guardando la informacion clave (id de la sala, email, inicio y fianal). Permitiendo que al momento de que la persona ingrese al resinto muestre el qr permitiendo un ingreso mas seguro y eficaz.
